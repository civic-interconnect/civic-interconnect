# Context Tags

Context tags define the scope under which observations or explanations apply.

They constrain interpretation without altering underlying data.

---

## Purpose

Context tags express:

-   Jurisdictional scope
-   Temporal applicability
-   Methodological assumptions
-   Normative framing

They make implicit assumptions explicit.

---

## Attachment

Context tags may be attached to:

-   Observations
-   Explanations
-   Models

They do not attach to raw records.

---

## Neutrality

Context tags do not assert correctness.

They provide clarity about applicability.
